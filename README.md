# AsikpitX-UserBot

A simple Python Telegram User Bot using **Telethon**.

## Features
- Responds to `/start` and `/ping` commands
- Easy to set up and run
- Ready to deploy on Replit

## Installation (Local)

1. Clone this repository:
```bash
git clone https://github.com/AsikpitX/AsikpitX-UserBot.git
cd AsikpitX-UserBot

2. Install dependencies:
pip install -r requirements.txt

3. Create .env file with your credentials:

API_ID=YOUR_API_ID
API_HASH=YOUR_API_HASH

4. Run the bot:

python bot.py

Deploy on Replit

1. Go to Replit and login/signup.


2. Click Create → Import from GitHub.


3. Paste your repo URL: https://github.com/AsikpitX/AsikpitX-UserBot


4. Set Secrets for your environment variables:

Key: API_ID, Value: your_api_id

Key: API_HASH, Value: your_api_hash



5. Set Run command in .replit file:



run = "python bot.py"

6. Click Run and your bot will start.



> ⚠️ Free Replit bots may go to sleep. Use a service like UptimeRobot to ping the bot periodically for 24/7 uptime.
